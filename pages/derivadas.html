Regras de Derivação

Potência

d/dx [xⁿ] = n·xⁿ⁻¹

d/dx [x³] = 3x²

Seno

d/dx [sen(x)] = cos(x)

d/dx [sen(0)] = 1

Cosseno

d/dx [cos(x)] = -sen(x)

d/dx [cos(0)] = 0

Exponencial

d/dx [eˣ] = eˣ

d/dx [e¹] = e ≈ 2.718

Logaritmo

d/dx [ln(x)] = 1/x

d/dx [ln(2)] = 0.5

Constante

d/dx [c] = 0

d/dx [5] = 0

Exercícios

Exercício 1

f(x) = x², f'(2) = ?

f'(x) = 2x
f'(2) = 2·2 = 4

Exercício 2

f(x) = sen(x), f'(0) = ?

f'(x) = cos(x)
f'(0) = cos(0) = 1

Exercício 3

f(x) = eˣ, f'(1) = ?

f'(x) = eˣ
f'(1) = e ≈ 2.718

Exercício 4

f(x) = ln(x), f'(2) = ?

f'(x) = 1/x
f'(2) = 1/2 = 0.5

Dicas Rápidas

  • A derivada é a inclinação da reta tangente
  • No gráfico, a reta roxa tracejada mostra a tangente
  • Se f'(x) > 0, a função é crescente
  • Se f'(x) < 0, a função é decrescente
  • Se f'(x) = 0, pode ser ponto de máximo/mínimo
